Téléchargement de document

Définition :

l'action de télécharger un document signifie que le document est accessible sur l'ordinateur sans avoir à se connecter à internet.

Définition :

Lorsqu'un lien est cliqué bien souvent le document s'ouvre à l'intérieur du navigateur (utilisant un plugin ou module d'extension). Il n'est pas alors disponible en dehors d'internet. Lorsque le document est laissé volontairement sur internet pour être téléchargé un clic sur le document avec le bouton droit et en sélectionnant l'article "enregistrer la cible sous" permet de sauvegarder le document.

Certains module d'extension permette la sauvegarde du document.
